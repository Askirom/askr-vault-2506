# Device Type Codes

  A consistent short-code system for naming devices.
  Each code is exactly two letters, lowercase, to ensure cross-platform compatibility.

## Audio

- hp :: Headphones
- sp :: Speakers
- mc :: Microphone

## Computers & Tablets


- pc :: Desktop PC
- tb :: Tablet 

## Phones & Wearables

- ph :: Phone (smartphone)
- sw :: Smartwatch
- fb :: Fitness Band

## Peripherals

- kb :: Keyboard
- ms :: Mouse
- mn :: Monitor
- pr :: Printer

## Networking & Storage

- rt :: Router
- ns :: NAS (Network Attached Storage)
- ex :: External Drive

## Consoles & Gaming

- sd :: Steam Deck
- ps :: PlayStation
- xb :: Xbox
- ns :: Nintendo Switch

## Smart Home & IoT

- lt :: Smart Light
- cm :: Camera
- ss :: Smart Sensor

## Miscellaneous

- dc :: Docking Station
- ch :: Charger
